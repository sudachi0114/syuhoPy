# 週報自動送信プログラム

## Usage;

* **Only first-time** 

```
cp config.sample.json config.json
```

* **To Change Email address configuration** please replace Here ⬇️.

```json:config.json
{
	// please change 2 email address below, for your condition
	"to_addr": "to@gmail.com",
	"from_addr": "from@gmail.com"
}
```
